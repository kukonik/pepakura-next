# Pepakura Next

Pepakura Next — кроссплатформенный инструмент нового поколения для развёртки 3D‑моделей в бумажные шаблоны и работы с 2D/3D‑контентом, вдохновлённый классическим Pepakura Designer и современными AI‑подходами к генерации и оптимизации моделей. [web:373][web:375]

## Возможности

- Кроссплатформенное десктоп‑приложение на базе Tauri 2 + Vue 3 + Rust.
- Гибридная архитектура: десктоп и веб‑клиент с общими модулями и форматом проекта.
- Пайплайн Text → 2D → 3D → бумажная развёртка:
  - генерация 2D‑эскизов и референсов по тексту (планируется);
  - реконструкция и упрощение 3D‑моделей;
  - развёртка 3D‑форм в 2D‑шаблоны с клапанами и линиями сгиба. [web:342][web:345]
- Экспорт развёртки в PDF (одна страница на каждый лист с контурами деталей).
- Проектный формат для сохранения состояния на любом этапе пайплайна.

## Текущий статус

Проект находится в активной разработке. Уже реализовано:

- Базовое десктоп‑приложение на Tauri 2.
- Импорт 3D‑модели и сохранение/загрузка проекта.
- Заглушка движка развёртки, возвращающая структуру листов и деталей.
- Простая визуализация листов и деталей в UI.
- Экспорт развёртки в PDF через встроенный Rust‑движок.

Планируется:

- Интеграция реального движка развёртки (Rust/Python).
- Поддержка SVG/PNG‑экспорта, продвинутые настройки линий и клапанов.
- AI‑модули для Text→2D→3D, автоматической оптимизации моделей и раскладки деталей.

## Стек

- **Desktop**: Tauri 2, Rust 2021, `serde`, `serde_json`, `printpdf`. [web:243]
- **Frontend**: Vue 3, TypeScript, Pinia.
- **Addons**: Python‑скрипты, вызываемые через отдельный процесс.
- **Контроль версий**: Git + GitHub.

## Сборка и запуск (desktop)

Требования:

- Node.js (LTS) и pnpm. [web:310]
- Rust (через rustup) и необходимый MSVC toolchain.
- Git.

Команды:

```bash
cd ui-desktop
pnpm install
pnpm tauri dev
```

Для сборки релизной версии:

```bash
cd ui-desktop
pnpm tauri build
```
